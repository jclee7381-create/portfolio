GitHub Pages 배포 방법

  1. 이 폴더 전체를 새 GitHub 저장소의 루트에 업로드합니다.
  2. 저장소 설정에서 Pages를 열고 배포 브랜치를 main, 폴더를 /(root)로 선택합니다.
  3. 몇 분 뒤 생성된 주소로 접속하면 루트 메인 페이지와 3개 프로젝트가 함께 열립니다.
  4. 각 프로젝트 이미지는 해당 폴더의 images 안에 넣으면 바로 교체됩니다.
